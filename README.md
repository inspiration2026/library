# Book Library Tracker
A simple JavaScript book library application where users can add books, mark them as read/unread, delete entries, and view their collection in a clean table.
Educational project built while passing JavaScript course in Odin Project.

For educational purposes project build entirely in JS with empty CSS file and empty HTML file Body.

## Features
- Add new books via a dynamic form (title, author, pages, read status)
- Display all books in a responsive table
- Toggle "read" status with one click (YES / NO)
- Delete books from the list
- Prevent adding duplicate books (same title + author + pages)
- Unique ID for each book using `crypto.randomUUID()`
- Clean table styling with auto-scroll when many books are added
- No external libraries — HTML with empty body, empty CSS, only JavaScript

## Method used

- `crypto.randomUUID()` for unique IDs
- DOM manipulation & event delegation patterns
- Styling of the elements was made by two different ways on purpose
